Class BasicMemoryCacheWrapperFactory

Creates and initializes instances of IMemoryCacheWrapper<T> This class should not be used when Dependency Injection is used as it directly new up instances of IMemoryCacheWrapper<T> rather than resolving them from the container. For a DI compatible solution look for a factory implementation in a Ring 2 library like VaraniumSharp.Initiator
