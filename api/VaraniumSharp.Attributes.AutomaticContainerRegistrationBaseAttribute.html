MultipleConstructors

Indicate if the service has multiple constructors. Containers like DryIoC need this in order to be able to properly register classes that have more than one constructor. https://bitbucket.org/dadhi/dryioc/wiki/SelectConstructorOrFactoryMethod#markdown-header-selecting-constructor-with-resolvable-parameters

If the implementing container does not need it, this property can be ignored
